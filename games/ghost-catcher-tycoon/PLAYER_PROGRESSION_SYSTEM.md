# Player Progression System — Ghost Catcher Tycoon

## Overview

Players progress through **100 Levels** by catching ghosts and completing quests. Higher levels unlock better equipment, new zones, and higher rewards.

---

## Level System (1-100)

### XP Requirements

```
Level 1-10:    100 XP per level
Level 11-25:   250 XP per level
Level 26-50:   500 XP per level
Level 51-75:   1,000 XP per level
Level 76-100:  2,500 XP per level

Total XP to reach Level 100: ~175,000 XP
```

### Level Rewards

Every level grants:
- **1 Skill Point** — Allocate to passive bonuses
- **Base Coins** — 100 × Level (Level 10 = 1,000 bonus coins)
- **Unlock Milestone** — Every 5 levels unlocks new equipment or zone

## Skill Points & Passive Bonuses

Each level grants **1 Skill Point**. Players allocate to boost permanent stats.

### Available Skills (Choose 1 per point)

#### Catching Bonuses
- **Better Net** (+2% catch rate, cost: 5 points)
- **Faster Charge** (-0.5s charge time, cost: 3 points)
- **Steady Aim** (+3% accuracy on moving ghosts, cost: 4 points)

#### Energy & Stamina
- **Energy Pool** (+5 max energy, cost: 2 points)
- **Energy Regen** (+1 energy/sec, cost: 3 points)
- **Power Surge** (+10% damage on next 3 catches, cost: 5 points)

#### Coins & Rewards
- **Greed** (+5% coins earned, cost: 3 points)
- **Jackpot** (+10% rare ghost spawn rate, cost: 4 points)
- **Bonus XP** (+10% XP earned, cost: 3 points)

#### Movement & Utility
- **Speed Burst** (+20% run speed for 5s, recharge 30s, cost: 4 points)
- **Ghost Sense** (see ghosts 50 studs away on minimap, cost: 5 points)
- **Lucky Find** (+chance to find bonus coins on ground, cost: 2 points)

## Quest System

### Daily Quests (Reset every 24 hours)

Available from **Level 25+**

**Quest Examples:**
- "Catch 5 Common ghosts" → Reward: 500 coins + 100 XP
- "Catch 2 Rare ghosts" → Reward: 2,000 coins + 300 XP
- "Catch a ghost in each zone" → Reward: 1,000 coins + 250 XP
- "Earn 10,000 coins" → Reward: Bonus 2,000 coins + 200 XP
- "Catch 1 Legendary ghost" → Reward: 5,000 coins + 500 XP

**Daily Streak Bonus:**
- Complete quest 3 days in a row → +50% reward next quest
- 7 days in a row → Unlock special cosmetic reward
- 30 days in a row → Title: "Dedication"

### Weekly Quests (Reset every 7 days)

Available from **Level 40+**

**Quest Examples:**
- "Catch 50 ghosts total" → Reward: 10,000 coins + 1,000 XP
- "Collect 100,000 coins" → Reward: Bonus 15,000 coins + 1,500 XP
- "Reach Level +5 from week start" → Reward: Special equipment skin + 2,000 XP
- "Catch 5 different Legendary ghosts" → Reward: 25,000 coins + 3,000 XP

### Challenge Quests (Available anytime, can be active)

Available from **Level 35+**

**One-time high-difficulty quests:**
- "Catch 10 Corrupted ghosts without dying" → Reward: 50,000 coins + 5,000 XP + Rare Cosmetic
- "Catch a ghost in all 12 zones in one session" → Reward: 30,000 coins + 3,000 XP + Weapon Skin
- "Beat personal record: catch most ghosts in 1 hour" → Reward: Leaderboard ranking + Season cosmetic

## Prestige System (Level 80+)

Once you reach **Level 80**, you unlock **Prestige Mode**.

### How Prestige Works

1. **Soft Reset:** Keep all equipment, lose levels, reset to Level 1
2. **Prestige Bonus:** Earn +10% XP on second playthrough, +20% on third, etc.
3. **Prestige Title:** "Prestige 1", "Prestige 2" etc. shown next to player name
4. **Cosmetics:** Unlock special skins and titles per Prestige level
5. **Prestige Rewards:** Every prestige grants exclusive cosmetics/perks

### Prestige Levels

- **Prestige 1** → +10% XP, +5% coins, Gold glow effect
- **Prestige 2** → +20% XP, +10% coins, Purple glow effect
- **Prestige 3+** → +30% XP, +15% coins, Rainbow glow effect + custom title

## Cosmetics & Rewards

### Account-Wide Cosmetics
- **Equipment Skins** — Reskin your net/trap/blaster
- **Ghost Capture Effects** — Particles, sounds, animations on successful catch
- **Player Skins** — Outfit/character appearance
- **Titles** — Shown next to player name: "Ghost Hunter", "Legend", "Corrupted Slayer"
- **Emotes** — Animations to perform in-game

### How to Earn Cosmetics
- **Level Milestones** → Automatic unlock
- **Quest Completion** → Specific quest rewards
- **Prestige Resets** → Exclusive prestige cosmetics
- **Battle Pass** (future) → Seasonal cosmetics for active play

## XP Rewards by Ghost Rarity

### Base XP (before bonuses)

| Rarity | Base XP | Zone Multiplier |
|---|---|---|
| Common | 25 | 1.0× (early) → 2.0× (late) |
| Uncommon | 50 | 1.2× |
| Rare | 100 | 1.5× |
| Epic | 200 | 2.0× |
| Legendary | 500 | 2.5× |
| Corrupted | 1,000 | 3.0× |

**Zone Multiplier:** Harder zones give more XP for same ghost

**Example:**
- Catch Common ghost in Whisper Woods (easy): 25 XP
- Catch Common ghost in The Rift (hard): 50 XP
- Catch Legendary ghost in Eternity Nexus (hardest): 500 × 2.5 = 1,250 XP

## Progression Timeline Example

### New Player (Level 1)
- Start with Basic Net
- Catch Common ghosts in Whisper Woods
- Earn ~25 XP + 50 coins per ghost
- Need ~4 ghosts to level up

### Early Player (Level 10)
- Unlock Ghost Trap equipment
- Unlock Foggy Fields zone
- Catching Uncommon ghosts
- Earning ~100 XP + 100 coins per ghost
- Unlocking first Skill Points

### Mid Player (Level 40)
- Unlock Proton Pack equipment
- Access to most zones
- Farming Rare/Epic ghosts
- Earning ~500+ XP + 500+ coins per ghost
- Unlocking Challenge Mode

### Veteran (Level 75)
- Ultimate equipment available
- Access all zones
- Can catch Legendary ghosts reliably
- Earning 1,000+ XP + 1,500+ coins per ghost
- Ready for Prestige reset

### Endgame (Level 100)
- Prestige rank achieved
- All cosmetics/titles unlocked
- Infinite Mode available
- Can tackle any ghost type
- Focus: High scores, leaderboards, cosmetics
