❌ COMMON ISSUES & FIXES:
════════════════════════════════════════════════════════════════════════════════

Issue: "No ghosts spawning"
Fix: Make sure both GhostAI and PhaseManager were created
     Check Output for: [PHASE 4] Spawn cycle messages

Issue: "Module not found" error
Fix: Check module name spelling (exactly: GhostAI, PhaseManager)
     Verify they're ModuleScripts (not Scripts)
     Verify they're in ServerScriptService (not in a subfolder)

Issue: "Ghosts appear but don't move"
Fix: Verify GhostAI.lua was copied completely (should be ~195 lines)
     Check for red errors in Output

Issue: "Can't catch ghost"
Fix: Move closer to ghost (within 20 studs)
     Make sure you clicked CHARGE first
     Click CATCH when standing next to ghost

Issue: "Game crashes on startup"
Fix: Check Output for red Lua errors
     Most likely: One of the modules has incorrect code
     Re-copy the entire file (make sure you got everything)
